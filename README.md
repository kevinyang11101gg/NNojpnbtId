# Python高校教室管理系统

## 核心功能
- 首页
- 个人中心
- 教师管理
- 学生管理
- 教室信息管理
- 教师申请管理
- 学生申请管理
- 课时表管理
- 教师取消预约管理
- 学生取消预约管理

## 适用场景
- 高校教室资源管理
- 教师和学生预约教室
- 教学安排和课时表管理

## 技术特点
- 采用Python语言开发
- 使用MySQL数据库存储数据
- 基于 B/S 架构
- 利用Django框架构建
